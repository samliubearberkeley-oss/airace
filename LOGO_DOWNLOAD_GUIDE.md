## Recommended Online Tools

1. **SVG to PNG Converter**: https://cloudconvert.com/svg-to-png
2. **Logo Finder**: https://logos-world.net/ (has many brand logos)
3. **Simple Icons**: https://simpleicons.org/ (SVG format, need conversion)
4. **Wikimedia Commons**: https://commons.wikimedia.org/ (various formats)

## Steps to Get Logos

1. Visit the links above
2. Download SVG or PNG files
3. If SVG, convert to PNG (128x128px recommended)
4. Ensure transparent background
5. Save to `public/logos/` directory
6. Update `src/lib/insforge.ts` to use logo paths instead of emojis

## File Naming Convention

Save logos as:
- `openai.png` or `gpt-4o.png`, `gpt-5.png`
- `anthropic.png` or `claude.png`
- `google.png` or `gemini.png`
- `xai.png` or `grok.png`
